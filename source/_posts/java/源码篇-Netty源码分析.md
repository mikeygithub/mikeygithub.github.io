# 框架特性

- 适用于各种传输类型的统一API-阻塞和非阻塞套接字
- 基于灵活且可扩展的事件模型，可将关注点明确分离
- 高度可定制的线程模型-单线程，一个或多个线程池，例如SEDA
- 真正的无连接数据报套接字支持（从3.1开始）



- 零拷贝
- NIO
